# gittivity
Mac desktop notifier for git browser activity

![demo](demo.png)

## Installation
Make sure you have `pip` install.
```
pip install gittivity
```

## Quick Start
To start gittivity- command
```
gittivity-start
```
To config gittivity- command
```
gittivity-config
```
To stop gittivity- command
```
gittivity-stop
```
For Help
```
gittivity-help
```
